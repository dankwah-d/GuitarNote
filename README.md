<h1>GuitarNotebk<h1>
<h2>A quick reference for tabs</h2>

<p>The purpose of this website is to help people learn guitar,
by providing a quick reference for chords.
Users have the ability to
<ul>  
<li>write notes for their technique </li>  
<li>Search chords</li> 
<li>Save favorite chords</li>
</ul>
